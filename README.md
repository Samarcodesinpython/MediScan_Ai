# 🩺 MediScan AI: Personalized Healthcare Intelligence Platform

An advanced, full-stack AI platform designed to provide personalized medical insights and services using machine learning and natural language processing.

## ✨ Core Features

| Feature | Technology | Description |
| :--- | :--- | :--- |
| **Disease Prediction** | **RandomForest** Classification | Predicts potential diseases based on user input (symptoms, history). |
| **Drug Alternatives** | **NLP** & **Cosine Similarity** | Recommends alternative medications based on existing drug profiles. |
| **Heart Risk Assessment** | **Machine Learning Models** | Assesses a user's risk level for heart disease based on health metrics. |
| **AI Medical Chatbot** | **OpenAI GPT-5** | Conversational interface for medical inquiries, health tips, and analysis. |

---

## 💻 Tech Stack

This is a comprehensive full-stack application built for performance, scalability, and type safety.

### 🚀 Frontend (Client)

* **Framework:** **React 18** & **TypeScript**
* **Build Tool:** **Vite** (for lightning-fast development)
* **UI/Design:** **shadcn/ui** (built on Radix UI) & **Tailwind CSS**
* **State Management:** **TanStack Query (React Query)** for server state
* **Forms:** **React Hook Form** with **Zod** validation

### ⚙️ Backend (Server)

* **Runtime:** **Node.js**
* **Framework:** **Express.js** & **TypeScript**
* **Database:** **PostgreSQL** 
* **ORM:** **Drizzle ORM** (for type-safe operations)
* **AI Integration:** **OpenAI API** (GPT-5)

---

## 🛠️ Getting Started

### Prerequisites

You need the following installed locally:

* [Node.js (v20+)]
* [npm]
* A **PostgreSQL** database (or use the provided Neon setup)
* An **OpenAI API Key**

### 1. Installation

Clone the repository and install dependencies for both the client and server:

```bash
git clone <github.com/saurabhbhandariii/MEDISCANAAI>

cd MediScanAI
npm install



```
